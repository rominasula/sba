# Your Travel Adventure

This is my SBA 307: HTML and CSS project.  
It’s a simple travel-themed website with *3 pages*: Home, Destinations, and Contact.

---

## Features
- 3 linked HTML pages (consistent layout)
- Table with travel destinations
- 2 forms (contact form + newsletter signup with dropdown)
- Inline, internal, and external CSS
- Images, GIFs, and text
- Simple navigation menu
- CSS animations (fade-in + hover effect)

---

## Pages
- *index.html* → Homepage with welcome text, images, and a table  
- *about.html* → Destination cards with pictures  
- *contact.html* → Contact form + newsletter signup  

---

## Technologies Used
- HTML5  
- CSS3
